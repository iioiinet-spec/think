집합 기초

원소·표기법·부분집합·공집합·원소 개수

📖 집합(Set)의 정의

집합이란 어떤 조건을 만족하는 서로 다른 대상들의 명확한 모임입니다. 집합에 속하는 각 대상을 원소(Element)라 합니다.

✅ 집합 O — 명확한 기준
  • 10 이하의 소수 → {2,3,5,7}
  • 대한민국 광역시 6개
  • 양의 짝수 {2,4,6,…}
  • 알파벳 모음 {a,e,i,o,u}
❌ 집합 X — 모호한 기준
  • 키가 큰 학생 (기준 불명확)
  • 맛있는 과일 (주관적)
  • 아름다운 수 (정의 불명확)
  • 거의 모든 짝수
핵심 규칙: 집합의 원소는 중복 없음, 순서 무관
{1,2,3} = {3,1,2} = {2,3,1}  |  {1,1,2,3} → 자동으로 {1,2,3}
✍️ 집합의 세 가지 표기법
표기법예시특징
원소나열법A = {1, 2, 3, 4, 5}원소를 직접 나열. 유한집합에 적합
조건제시법A = {x | x∈ℕ, x≤5}조건으로 정의. 무한집합도 표현 가능
벤다이어그램원 안에 원소 표시집합 간 관계를 시각화
⚙️ 원소 판별기 — x ∈ A ?

🔲 부분집합(Subset) & 진부분집합
A ⊆ B  ⟺  ∀x ( x∈A → x∈B )

집합 A의 모든 원소가 집합 B에 속하면 A는 B의 부분집합입니다.

핵심 성질:
· ∅ ⊆ A — 공집합은 모든 집합의 부분집합
· A ⊆ A — 자기 자신도 부분집합
· n개 원소 집합의 부분집합 수 = 2ⁿ
· A⊆B 이고 B⊆A 이면 A = B
진부분집합 A ⊂ B:
A ⊆ B 이고 A ≠ B

예) A={1,2}, B={1,2,3}
· A ⊂ B ✅   B ⊂ A ❌

성질결과

부분집합 수 = 2³ = 8개

∅, {a}, {b}, {c},

🔢 집합의 같음(Equality)
A = B  ⟺  A⊆B 이고 B⊆A

두 집합이 완전히 동일한 원소를 가질 때 같다고 합니다. 원소의 순서와 중복은 무시합니다.

{1,2,3} = {3,2,1} = {1,1,2,3}  → 모두 같은 집합!

집합 연산

합집합·교집합·차집합·여집합·대칭차집합

🧮 인터랙티브 집합 연산 계산기
집합 입력 후 연산 버튼 클릭
연산 버튼을 클릭하세요.
∪ 합집합 (Union)
A∪B = {x | x∈A 또는 x∈B}
A 또는 B 중 하나라도 속하면 포함
예) {1,2,3}∪{3,4,5} = {1,2,3,4,5}
∩ 교집합 (Intersection)
A∩B = {x | x∈A 이고 x∈B}
A와 B 모두에 속하는 원소만
예) {1,2,3}∩{3,4,5} = {3}
− 차집합 (Difference)
A−B = {x | x∈A 이고 x∉B}
A에 속하지만 B에는 없는 원소
예) {1,2,3}−{3,4,5} = {1,2}
ᶜ 여집합 (Complement)
Aᶜ = U−A = {x∈U | x∉A}
전체집합 U에서 A를 제외한 것
예) U={1..5}, A={1,3,5} → Aᶜ = {2,4}
△ 대칭차집합 (Symmetric Difference)
A △ B = (A−B) ∪ (B−A) = (A∪B) − (A∩B)

A 또는 B 중 하나에만 속하는 원소들의 집합입니다.

예) A={1,2,3}, B={3,4,5}
A△B = {1,2} ∪ {4,5} = {1,2,4,5}

집합의 법칙

교환·결합·분배·항등·보원·흡수·멱등

📋 집합 법칙 총정리
법칙합집합 (∪)교집합 (∩)
교환 법칙A∪B = B∪AA∩B = B∩A
결합 법칙(A∪B)∪C = A∪(B∪C)(A∩B)∩C = A∩(B∩C)
분배 법칙A∪(B∩C)=(A∪B)∩(A∪C)A∩(B∪C)=(A∩B)∪(A∩C)
항등 법칙A∪∅ = AA∩U = A
보원 법칙A∪Aᶜ = UA∩Aᶜ = ∅
흡수 법칙A∪(A∩B) = AA∩(A∪B) = A
멱등 법칙A∪A = AA∩A = A
드모르간(A∪B)ᶜ = Aᶜ∩Bᶜ(A∩B)ᶜ = Aᶜ∪Bᶜ
이중 여집합(Aᶜ)ᶜ = A
U·∅ 법칙A∪U = UA∩∅ = ∅
🔬 분배 법칙 실시간 검증기
세 집합 입력 후 분배법칙 성립 여부 확인
💡 흡수 법칙 증명
A ∪ (A∩B) = A
A∩B ⊆ A 이므로, A에 A∩B를 합해도 새 원소가 없음.
∴ A∪(A∩B) = A ▢
A ∩ (A∪B) = A
A ⊆ A∪B 이므로, A와 A∪B의 교집합 = A.
∴ A∩(A∪B) = A ▢

드모르간 법칙

여집합이 합집합·교집합을 서로 바꾼다

📖 드모르간 법칙 (De Morgan's Laws)
(A ∪ B)ᶜ = Aᶜ ∩ Bᶜ
(A ∩ B)ᶜ = Aᶜ ∪ Bᶜ
핵심 직관: 여집합을 분배하면 ∪ ↔ ∩ 으로 바뀐다!
프로그래밍 대응:
!(A || B) == !A && !B
!(A && B) == !A || !B

KOI 논리 조건 변환에 자주 사용!
집합 예시:
U={1..7}, A={1,2,3}, B={2,3,4}
(A∪B)ᶜ = {1..7}−{1,2,3,4} = {5,6,7}
Aᶜ∩Bᶜ = {4..7}∩{1,5,6,7} = {5,6,7} ✅
🎨 벤다이어그램 시각 증명
(A∪B)ᶜ
A∪B 밖의 영역
=
Aᶜ ∩ Bᶜ
두 여집합의 교집합
(A∩B)ᶜ
A∩B 밖의 영역
=
Aᶜ ∪ Bᶜ
두 여집합의 합집합
⚙️ 드모르간 수치 검증기
임의 집합으로 드모르간 법칙 확인
📐 형식 증명 — (A∪B)ᶜ = Aᶜ∩Bᶜ
[⊆ 방향] x∈(A∪B)ᶜ 가정:
  → x∉A∪B    (여집합 정의)
  → x∉A 이고 x∉B    (합집합 부정 = 둘 다 아님)
  → x∈Aᶜ 이고 x∈Bᶜ  → x∈Aᶜ∩Bᶜ ✓

[⊇ 방향] x∈Aᶜ∩Bᶜ 가정:
  → x∈Aᶜ 이고 x∈Bᶜ
  → x∉A 이고 x∉B  → x∉A∪B  → x∈(A∪B)ᶜ ✓

∴ (A∪B)ᶜ = Aᶜ∩Bᶜ    ▢

멱집합 · 곱집합

P(A) — 모든 부분집합  |  A×B — 순서쌍 집합

🔋 멱집합 (Power Set) P(A)
P(A) = {S | S ⊆ A}     |P(A)| = 2^|A|

집합 A의 모든 부분집합을 원소로 하는 집합. ∅와 A 자신 포함.

A = {a, b, c} → |A|=3
P(A) = { ∅, {a}, {b}, {c},
         {a,b}, {a,c}, {b,c},
         {a,b,c} }
|P(A)| = 2³ = 8
💡 비트마스크 연결:
n개 원소를 0~n-1 비트에 대응
i번째 원소 포함 ↔ i번 비트=1

부분집합 전체 열거:
for(int s=0;s<(1<<n);s++)
멱집합 생성기 (최대 5원소)
✖️ 곱집합 (Cartesian Product) A × B
A×B = {(a,b) | a∈A, b∈B}     |A×B| = |A|×|B|

A의 원소 a와 B의 원소 b로 만든 모든 순서쌍 (a,b)의 집합.

A={1,2,3}, B={x,y}
A×B = {(1,x),(1,y),(2,x),(2,y),(3,x),(3,y)}
|A×B| = 3 × 2 = 6
⚠ 주의:
A×B ≠ B×A (순서 다름)
(1,x) ≠ (x,1)

활용: 관계 정의 R⊆A×B,
좌표평면 ℝ×ℝ

포함-배제 원리

합집합의 원소 개수를 빠짐없이 세기

📖 두 집합 포함-배제
|A ∪ B| = |A| + |B| − |A ∩ B|
왜 빼야 하나? |A|+|B|를 더하면 A∩B에 속하는 원소가 두 번 계산됨. 중복된 |A∩B|를 한 번 빼준다.
두 집합 포함-배제 계산기
📖 세 집합 포함-배제
|A∪B∪C| = |A|+|B|+|C| − |A∩B| − |B∩C| − |A∩C| + |A∩B∩C|
단계연산이유
① 더하기+|A|, +|B|, +|C|각 집합 원소 수 합산
② 빼기−|A∩B|, −|B∩C|, −|A∩C|두 번 셈한 교집합 제거
③ 더하기+|A∩B∩C|세 번 더하고 세 번 뺀 것 보정
세 집합 포함-배제 계산기
🧮 KOI 응용 — 1~N에서 A 또는 B의 배수가 아닌 수의 개수
결과 = N − ⌊N/A⌋ − ⌊N/B⌋ + ⌊N/lcm(A,B)⌋
lcm(A,B) = A × B / gcd(A,B)    (유클리드 호제법으로 gcd 계산)

관계와 함수

이진관계의 성질 · 함수의 종류

📖 이진 관계 (Binary Relation)
R ⊆ A×B    (a,b)∈R  ⟺  "aRb"

집합 A에서 B로의 이진 관계는 A×B의 부분집합입니다.

A={1,2,3}, "x < y" 관계: R={(1,2),(1,3),(2,3)} ⊆ A×A
🔑 관계의 성질 (A→A 자기 관계)
성질정의예시
반사적 (Reflexive)∀a∈A: aRa≤ — 1≤1, 2≤2 성립
비반사적 (Irreflexive)∀a∈A: ¬(aRa)< — 1<1 불성립
대칭적 (Symmetric)aRb → bRa= — a=b이면 b=a
반대칭적 (Antisymmetric)aRb∧bRa → a=b≤ — a≤b∧b≤a → a=b
추이적 (Transitive)aRb∧bRc → aRc< — a<b<c → a<c
동치 관계: 반사적 + 대칭적 + 추이적
예: =, "같은 학교 출신"
편순서 관계: 반사적 + 반대칭적 + 추이적
예: ≤, ⊆
🎛 관계 행렬 인터랙티브 — 클릭으로 관계 정의

A={1,2,3,4} 위의 관계 R. 셀을 클릭하여 켜기/끄기. 성질 자동 판별.

🔢 함수 (Function) f: A → B
모든 a∈A에 대해 유일한 b∈B가 대응
단사 (Injective)
a₁≠a₂ → f(a₁)≠f(a₂)
서로 다른 원소 → 다른 상
전사 (Surjective)
∀b∈B, ∃a: f(a)=b
공역 모든 원소에 도달
전단사 (Bijective)
단사+전사
완전한 1:1 대응, 역함수 존재
