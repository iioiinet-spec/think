관계와 함수 문제집

1. 관계 (Relation) R ⊆ A×B

집합 A와 B의 곱집합 A×B의 부분집합을 관계라고 한다.

A = {1,2}, B = {a,b} A×B = {(1,a),(1,b),(2,a),(2,b)} R = {(1,a),(2,b)} → A×B의 부분집합이므로 관계이다.

2. 이진관계 (Binary Relation) R ⊆ A×A

A에서 A로 가는 관계, 즉 같은 집합 안에서 원소들 사이의 연결을 나타낸다.

A = {1,2,3} R = {(1,1),(2,2),(3,3)}

3. 이진관계의 4가지 성질 암기 필수

성질조건
반사성 (Reflexive)모든 a에 대해 (a,a) ∈ R
대칭성 (Symmetric)(a,b) ∈ R 이면 (b,a) ∈ R
반대칭성 (Antisymmetric)(a,b)와 (b,a)가 동시에 있으면 a=b (단 (a,a)는 허용)
추이성 (Transitive)(a,b)와 (b,c)가 있으면 (a,c)도 있어야 한다

4. 함수 (Function)와 함수의 종류

A의 모든 원소가 B의 정확히 하나의 원소에 대응할 때 함수라고 한다.

종류조건
함수모든 입력이 정확히 하나의 출력에 대응
일대일 함수 (Injective)서로 다른 입력은 서로 다른 출력에 대응
전사 함수 (Surjective)공역(B)의 모든 원소가 적어도 한 번 대응됨
전단사 함수 (Bijective)일대일 함수이면서 동시에 전사 함수

정보올림피아드 응용 포인트 심화

동치관계(등가관계) = 반사성 + 대칭성 + 추이성 순서관계(부분순서) = 반사성 + 반대칭성 + 추이성

이 개념들은 그래프, 위상정렬, 유니온 파인드, 집합 분할, DFS/BFS, 함수형 그래프 등 다양한 주제로 이어진다.
