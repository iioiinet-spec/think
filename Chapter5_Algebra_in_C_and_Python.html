대수와 함수를 코드로 구하기 💻

식 계산, 함수 만들기, 함수표·그래프 값까지! 값을 바꾸면 코드와 결과가 바로 바뀌어요.

🧰 준비물: 변수와 식

대수의 핵심은 '모르는 수를 글자로 두기'. 코드에서는 그 글자가 바로 변수예요!

📘 변수 = 수를 담는 상자
x = 3 은 "x라는 상자에 3을 담아라"예요.
그러면 y = 2 * x + 1 처럼 식에 x를 넣어 계산할 수 있어요!
CC 언어
// 식 y = 2x + 1 계산
int x = 3;
int y = 2 * x + 1;
printf("y=%d\n", y);  // 7
PyPython
# 식 y = 2x + 1 계산
x = 3
y = 2 * x + 1
print("y=", y)   # 7
🔬 식 계산기 — y = a×x + b
y = × x +
x =
▶ 결과
💡 곱하기 기호는 꼭!
수학에서는 2x라고 쓰지만, 코드에서는 곱하기 *를 꼭 써야 해요! 2 * x 처럼요.

🔮 함수 기계를 코드로 만들기

5장의 '함수 기계'를 진짜 코드 함수로! 수를 넣으면 규칙대로 답을 돌려줘요.

📘 함수 = 재사용하는 기계
한 번 만들면 이름만 불러서 계속 쓸 수 있어요.
C는 int f(int x){...}, Python은 def f(x): 로 만들어요.
🔬 코드 실험실 — 함수 f(x) = a×x + b
곱하기 a = 2
더하기 b = 1
넣는 x = 3

PyPython

🔑 핵심
함수를 한 번 만들면 f(3), f(5), f(10) 처럼 여러 번 불러 쓸 수 있어요. 기계를 재사용하는 거죠!

📋 함수표 만들기 (반복문)

x를 0,1,2,3… 차례로 넣어 y값을 모두 구하면 함수표! 반복문이 자동으로 만들어줘요.

🔬 코드 실험실 — y = a×x + b 표
곱하기 a = 2
더하기 b = 1
x 끝값 = 5
CC 언어
PyPython
▶ 실행 결과 (x → y)
💡 규칙이 보여요
표를 보면 x가 1 커질 때 y가 a씩 커져요! 코드로 표를 뽑으면 규칙이 한눈에 보인답니다.

📊 그래프 점 좌표 구하기

함수표의 (x, y)가 바로 그래프의 점! 좌표를 (x, y) 모양으로 출력해봐요.

📘 점 = (x, y)
x값마다 y를 구해 (x, y) 쌍으로 묶으면 그래프에 찍을 점이 돼요. 이 점들을 이으면 직선 그래프!
🔬 코드 실험실 — 점 좌표 출력
기울기 a = 1
더하기 b = 2
CC 언어
PyPython
▶ 실행 결과 (점 좌표)
🔑 핵심
Python의 f-string f"({x}, {y})" 로 좌표를 예쁘게 묶어 출력할 수 있어요!

🔍 숨은 수 □ 거꾸로 찾기

a×□ + b = 결과 일 때 □를 구하기! 코드로도 거꾸로 계산하면 돼요: □ = (결과 − b) ÷ a.

📘 거꾸로 식
a × □ + b = c 라면 → □ = (c − b) / a
더하기는 빼기로, 곱하기는 나누기로 거꾸로 풀어요!
🔬 코드 실험실 — □ 구하기
곱하기 a = 2
더하기 b = 3
결과 c = 11
CC 언어
PyPython
▶ 실행 결과
💡 나누어떨어지나 확인
(c − b)가 a로 딱 나누어떨어져야 □가 깔끔한 정수예요! 안 떨어지면 답이 소수가 돼요.

⚖️ C 와 Python, 대수·함수 코드 비교

변수·식·함수에서 자주 쓰는 표현을 정리했어요. Python 함수가 얼마나 짧은지 비교해봐요!

하고 싶은 것C 언어Python
변수 만들기int x = 3;x = 3
식 계산y = 2 * x + 1;y = 2 * x + 1
함수 정의int f(int x){ return ...; }def f(x): return ...
함수 호출f(3)f(3)
좌표 출력printf("(%d,%d)", x, y);print(f"({x},{y})")
소수 나누기(double)(c-b)/a(c-b)/a
📘 함수 한 줄 비교
같은 함수도 C는 자료형(int)과 return·중괄호가 필요하고, Python은 def f(x): return 2*x+1 한 줄이면 돼요!
⚠️ 곱하기 기호 빼먹지 않기
수학의 2x, 3(x+1) 은 코드에서 2*x, 3*(x+1) 로 곱하기를 꼭 써야 해요. 안 쓰면 오류가 나요!
