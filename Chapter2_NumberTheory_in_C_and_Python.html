정수론을 코드로 구하기 💻

손으로 찾던 약수·소수·최대공약수를 C와 Python으로! 값을 바꾸면 코드와 결과가 바로 바뀌어요.

🧰 준비물: 나머지 연산 %

정수론 코드의 비밀 무기는 나머지(%)예요. "나누어떨어지는가?"를 컴퓨터가 판단하는 방법이죠.

📘 나머지 연산자 %
a % b 는 a를 b로 나눈 나머지예요.
그래서 a % b == 0 이면 "a는 b로 나누어떨어진다 = b는 a의 약수"라는 뜻!
CC 언어
// 12를 5로 나눈 나머지
printf("%d\n", 12 % 5);  // 2

// 나누어떨어지는지 확인
if (12 % 3 == 0)
    printf("3은 12의 약수!\n");
PyPython
# 12를 5로 나눈 나머지
print(12 % 5)   # 2

# 나누어떨어지는지 확인
if 12 % 3 == 0:
    print("3은 12의 약수!")

🔑 체의 원리
"모두 소수다"라고 가정한 표(isPrime)에서, 소수를 찾을 때마다 그 배수를 전부 거짓으로 지워요. 끝까지 참으로 남은 수가 소수!

🤝 최대공약수 · 최소공배수

큰 수를 작은 수로 나눈 나머지로 계속 바꿔치기! '유클리드 호제법'이라는 똑똑한 방법이에요.

📘 유클리드 호제법
두 수의 최대공약수는 (큰 수 ÷ 작은 수의 나머지)로 계속 바꾸다가, 나머지가 0이 되면 그때의 수가 답!
예: GCD(18,12) → GCD(12,6) → GCD(6,0) → 6. 그리고 최소공배수 = a × b ÷ 최대공약수.

b = 12

📘 함수로 묶으면 편해요
약수 구하기·소수 판별처럼 자주 쓰는 일은 함수로 만들어두면 이름만 불러서 재사용할 수 있어요. C는 int isPrime(int n){...}, Python은 def is_prime(n): 처럼요.
⚠️ 정올 꿀팁
소수 판별은 i*i <= n 까지만, 소수 목록은 에라토스테네스의 체를 쓰면 훨씬 빨라요. 문제에서 수가 아주 크면 빠른 방법이 꼭 필요해요!
