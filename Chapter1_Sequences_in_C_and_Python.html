수열을 코드로 출력하기 💻

손으로 찾은 규칙을 C와 Python으로! 두 언어를 나란히 보고, 값을 바꾸면 결과가 바로 바뀌어요.

🧰 준비물: 출력하기와 반복하기

수열을 코드로 만들려면 딱 두 가지면 돼요 — ① 화면에 출력하기, ② 여러 번 반복하기.

📘 ① 출력하기
C는 printf, Python은 print 로 화면에 글자나 수를 보여줘요.
CC 언어
// 안녕 출력하기
#include <stdio.h>
int main() {
    printf("안녕 수열!\n");
    return 0;
}
PyPython
# 안녕 출력하기
print("안녕 수열!")
📘 ② 반복하기 (for)
같은 일을 여러 번 시킬 때 for 반복문을 써요. "0번부터 n번까지 돌아라"는 뜻이에요.
CC 언어
for (int i = 0; i < 5; i++) {
    printf("%d ", i);
}
// 출력: 0 1 2 3 4
PyPython
for i in range(5):
    print(i, end=" ")
# 출력: 0 1 2 3 4
💡 기억!
C는 문장 끝에 세미콜론(;)과 중괄호 { }가 필요하고, Python은 들여쓰기(띄어쓰기)로 묶어요. %d는 C에서 "여기에 정수를 넣어라"는 뜻이에요.

⚖️ C 와 Python, 무엇이 다를까?

같은 수열을 만들어도 두 언어는 생김새가 달라요. 한 표로 정리해요.

📘 언제 무엇을?
Python은 짧고 쉬워서 아이디어를 빠르게 시험할 때 좋아요. C는 컴퓨터가 어떻게 계산하는지 자세히 배우고, 정보올림피아드(정올)처럼 빠른 속도가 필요할 때 강해요.
⚠️ C에서 조심
등비·피보나치는 수가 아주 빨리 커져요. C의 int는 너무 큰 수를 담지 못해요. 그럴 땐 long long을 쓰고, Python은 걱정 없이 아주 큰 수도 다뤄요!
