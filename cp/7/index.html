시작하기 — 문자열이란?

글자들이 순서대로 모인 것 — C와 파이썬은 속이 다르다

1문자열 = 문자들의 나열글자 + 글자

문자열(string)은 글자(문자)들이 순서대로 이어진 것이에요. "Hello", "안녕", "abc123" 모두 문자열이죠.

사실 문자열은 문자 배열이에요! 배열/리스트를 배웠다면 이미 절반은 안 셈이에요. 각 글자에 인덱스(0부터)가 붙어요.

"Hello" 의 내부 구조
0H
1e
2l
3l
4o

s[0]은 'H', s[2]는 'l'이에요. 문자열은 곧 문자 배열이라 인덱스로 한 글자씩 꺼낼 수 있어요!

2C의 비밀 — 널 종료 문자 '\0'끝을 알리는 표시

C에서 문자열은 char 배열이고, 맨 끝에 '\0'(널 문자)이 숨어있어요! 이게 "여기서 문자열 끝"이라는 종료 표시예요.

C: char s[] = "Hi"; (실제로는 3칸!)
0H
1i
2\0
⚠️ C의 널 종료 주의! "Hi"는 글자 2개지만 배열은 3칸이 필요해요('\0' 포함).
그래서 char s[6]에는 최대 5글자만 담을 수 있어요. 마지막 1칸은 항상 '\0' 몫!
3C vs 파이썬 — 핵심 차이배열 vs 객체
비교C 문자열파이썬 문자열
정체char 배열 + '\0'str 객체
만들기char s[] = "Hi";s = "Hi"
길이strlen(s)len(s)
이어붙이기strcat(a, b)a + b
비교strcmp(a, b)a == b
수정가능 (배열이라서)불가! (불변)
💡 가장 큰 차이 2가지: ① C는 string.h의 특수 함수(strlen, strcmp...)가 필요하지만, 파이썬은 +, ==, len()으로 자연스럽게 써요.
② 파이썬 문자열은 불변(immutable) — s[0]='X'가 안 돼요! 바꾸려면 새 문자열을 만들어야 해요.

기본 — 입력·출력·길이

문자열 받고, 출력하고, 길이 재기

1문자열 입력받기scanf vs input
C 언어파이썬
char s[100];
scanf("%s", s);
// 공백 전까지만!		 s = input()

# 한 줄 전체
⚠️ C의 scanf 함정! scanf("%s", s)는 공백을 만나면 멈춰요.
"Hello World"를 입력해도 "Hello"만 읽혀요! 공백 포함 한 줄은 fgets(s, 100, stdin)을 써야 해요.
또 scanf("%s", s)는 배열이라 &를 안 붙여요! (배열 이름이 곧 주소)
2길이 재기 — strlen vs len글자 수 세기

C의 strlen은 사실 '\0'을 만날 때까지 세는 반복문이에요! 파이썬 len은 바로 알려줘요.

🔵 C — strlen 내부 원리
int myLen(char s[]) { int i = 0; while (s[i] != '\0') { // 널 만날때까지 i++; } return i; // 센 개수 }
🟢 Python — len 내장
s = "Hello" print(len(s)) # 5 # 길이를 바로 알려줌!
💡 C 문자열의 길이를 모르면 '\0'까지 세야 알 수 있어요. 그래서 strlen은 문자열이 길수록 오래 걸려요. 파이썬 str은 길이를 미리 저장해둬서 즉시 알려줘요!
🖥 이름을 받아 인사 + 길이 출력
#include <stdio.h>

int main() {

    return 0;
}
# 이름 받아 인사 + 길이
name = input()
print(f"안녕, {name}!")
print(f"글자 수: {len(name)}")

문자 접근·순회

한 글자씩 꺼내기 · 아스키 코드 · 전체 훑기

1인덱스로 한 글자 꺼내기s[i]

문자열은 문자 배열이라 s[i]로 i번째 글자를 꺼내요. 배열/리스트와 똑같아요!

동작C파이썬
첫 글자s[0]s[0]
마지막 글자s[strlen(s)-1]s[-1] # 음수 인덱스!
부분 문자열직접 반복 복사s[1:4] # 슬라이싱!
💡 파이썬은 s[-1](마지막), s[1:4](슬라이싱) 같은 편리한 기능이 있어요! C에는 없어서 반복문으로 직접 만들어야 해요.
2아스키 코드 — 문자의 숫자값'A' = 65

컴퓨터는 문자를 숫자로 저장해요. 'A'=65, 'a'=97, '0'=48. 이걸 아스키 코드라 해요.

하고 싶은 것C파이썬
문자→숫자int n = s[i]; // 자동ord(s[i])
숫자→문자char c = 65; // 자동chr(65) # 'A'
대→소문자c + 32chr(ord(c)+32)
💡 C에서 문자는 사실상 숫자라 'A' + 1 == 'B'가 성립해요! 대문자에 32를 더하면 소문자가 되죠('A'+32='a'). 파이썬은 ord()/chr()로 변환해요.
🖥 문자열을 한 글자씩 세로로 출력 + 개수
#include <stdio.h>
#include <string.h>

int main() {
    char s[100];
    scanf("%s", s);
    int len = strlen(s);
    for (int i = 0; i < len; i++) {
        printf("[%d] %c\n", i, s[i]);  // 한 글자씩
    }
    printf("총 %d글자\n", len);
    return 0;
}
# 한 글자씩 세로 출력
s = input()
for i, ch in enumerate(s):  # 인덱스+문자
    print(f"[{i}] {ch}")
print(f"총 {len(s)}글자")

# for ch in s: 로 문자만 순회도 가능
C: for i<len인덱스로 s[i] 하나씩 접근
Py: enumerate(s)인덱스와 문자를 짝으로 순회! 아주 편리
%cC에서 문자 하나 출력 서식 (%s는 문자열)
단어를 넣으면 글자별로 인덱스와 함께 출력

문자열 함수

복사·비교·이어붙이기 — string.h vs 파이썬 연산자

1핵심 함수 4총사strlen·strcpy·strcmp·strcat

C는 문자열 작업에 string.h의 함수가 필요해요. 파이썬은 연산자로 자연스럽게!

하고 싶은 것C (string.h)파이썬
길이strlen(s)len(s)
복사strcpy(a, b)a = b
비교strcmp(a, b) == 0a == b
이어붙이기strcat(a, b)a + b
반복직접 반복문s * 3
⚠️ C에서 == 로 문자열 비교 금지! a == b는 주소를 비교해서 내용이 같아도 false가 나와요.
반드시 strcmp(a, b) == 0을 써야 "내용이 같다"를 확인해요. 초보자 최대 실수 중 하나!
2strcmp의 반환값 이해0 = 같음

strcmp(a, b)는 두 문자열을 사전 순서로 비교해서 숫자를 돌려줘요.

반환값의미
0두 문자열이 완전히 같음
음수 (<0)a가 사전순으로 앞 ("apple" < "banana")
양수 (>0)a가 사전순으로 뒤
💡 파이썬은 a == b, a < b가 그대로 문자열 비교예요! "apple" < "banana"는 True. C의 strcmp 반환값을 외우는 대신 직관적으로 쓸 수 있죠.
🖥 두 단어 비교 + 이어붙이기
#include <stdio.h>
#include <string.h>

int main() {
    char a[100], b[100];
    scanf("%s %s", a, b);
    // 비교
    if (strcmp(a, b) == 0)
        printf("같은 단어\n");
    else
        printf("다른 단어\n");
    // 이어붙이기
    strcat(a, b);          // a 뒤에 b 붙임
    printf("합치면: %s\n", a);
    return 0;
}
# 두 단어 비교 + 이어붙이기
a, b = input().split()

if a == b:              # 그냥 == !
    print("같은 단어")
else:
    print("다른 단어")

print("합치면:", a + b)  # + 로 연결!
C: strcmp(a,b)==0내용이 같으면 0 반환 (==는 주소비교라 금지!)
C: strcat(a,b)a 배열 뒤에 b를 이어붙임 (a가 바뀜)
Py: a==b, a+b비교도 연결도 연산자로 자연스럽게!
두 단어 (공백 구분). 같은 단어도 넣어보세요!

🖥 대문자로 변환 (아스키 코드 활용)
#include <stdio.h>
#include <string.h>

int main() {
    char s[100];
    scanf("%s", s);
    for (int i = 0; s[i]; i++) {
        if (s[i] >= 'a' && s[i] <= 'z')
            s[i] = s[i] - 32;  // 소→대 (아스키)
    }
    printf("%s\n", s);
    return 0;
}

🖥 문자열 뒤집기 (양끝 swap)
#include <stdio.h>
#include <string.h>

int main() {
    char s[100];
    scanf("%s", s);
    int left = 0, right = strlen(s) - 1;
    while (left < right) {   // 양끝 맞바꾸기
        char tmp = s[left];
        s[left] = s[right];
        s[right] = tmp;
        left++; right--;
    }
    printf("%s\n", s);
    return 0;
}
# 문자열 뒤집기
s = input()
print(s[::-1])    # 슬라이싱 한 방!

# [::-1]은 "거꾸로 전체" 슬라이싱
# reversed()로도 가능:
# print("".join(reversed(s)))
left=0, right=len-1양끝에서 시작하는 두 포인터
swap + 안쪽으로양끝을 바꾸며 가운데로 좁힘 (배열 reverse와 동일!)
Py: s[::-1]슬라이싱으로 거꾸로 — 놀랍도록 간단!
stressed → desserts! 회문(level)도 넣어보세요
