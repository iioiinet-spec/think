존슨 알고리즘 한 줄씩 따라가기

벨만–포드로 재가중한 뒤 다익스트라를 V번 — 부품들이 조립되는 모든 쌍 최단 경로

C 시각화: 포텐셜 h[] · 간선 재가중 · 다익스트라 V번

존슨 — 부품을 조립해 만드는 모든 쌍 최단 경로

플로이드–워셜은 모든 쌍 최단 거리를 O(V³)에 구합니다. 하지만 간선이 적은(희소) 그래프에서는 낭비가 커요. 다익스트라를 V번 돌리면 훨씬 빠르지만, 음수 간선이 있으면 다익스트라를 못 씁니다.

존슨 알고리즘은 이 딜레마를 기막히게 해결합니다. 벨만–포드로 그래프를 "재가중"해 음수를 없앤 뒤, 음수가 사라진 그래프에서 다익스트라를 V번 돌리는 거죠. 두 알고리즘을 부품으로 조립합니다.

플로이드–워셜존슨
음수 간선✅✅
복잡도O(V³)O(V²logV + VE)
희소 그래프불리유리
구성3중 for문벨만–포드 + 다익스트라 V번
💡 존슨의 천재성은 "음수를 없애되 최단 경로는 그대로 유지하는" 재가중에 있습니다. 이걸 벨만–포드가 계산한 포텐셜로 해냅니다. 오늘 그 마법을 단계별로 봅니다.

4단계 파이프라인

존슨 알고리즘은 네 단계로 흐릅니다. 각 단계가 어떤 부품을 쓰는지 보세요.

1

가상 정점 + 벨만–포드

가상 정점 0을 모든 정점에 0-가중으로 연결하고, 벨만–포드로 포텐셜 h[v]를 계산

벨만–포드
2

간선 재가중

w'(u,v) = w + h[u] − h[v] 로 모든 간선을 0 이상으로 변환

포텐셜 h[]
3

다익스트라 V번

음수가 사라진 재가중 그래프에서 각 정점을 출발점으로 다익스트라

다익스트라
4

거리 복원

d(u,v) = d'(u,v) − h[u] + h[v] 로 원래 거리로 되돌림

역변환
핵심 통찰 — 재가중은 거리의 절대값은 바꾸지만 어느 경로가 최단인지는 바꾸지 않습니다. 그래서 재가중 그래프에서 찾은 최단 경로가 원래 그래프에서도 최단이고, 거리만 공식으로 되돌리면 됩니다.

이 강의의 예제 그래프

정점 5개, 간선 9개인 가중치 유방향 그래프입니다(CLRS 고전 예제). 음수 간선이 여럿 있어요.

INPUT (u v w = u→v 방향, 가중치 w)
5 9 1 2 3 1 3 8 1 5 -4 2 4 1 2 5 7 3 2 4 4 1 2 4 3 -5 5 4 6
⚠️ 1→5가 −4, 4→3이 −5로 음수입니다. 다익스트라를 바로 못 쓰죠. 존슨은 이 음수들을 재가중으로 없앤 뒤 다익스트라를 씁니다.

포텐셜과 재가중 — 음수를 없애는 마법

각 정점 v에 포텐셜 h[v]라는 값을 부여합니다. 그리고 모든 간선을 이렇게 다시 매깁니다.

w'(u, v) = w(u, v) + h[u] − h[v]

이렇게 하면 놀라운 일이 벌어집니다. 어떤 경로든 그 길이가 h[시작] − h[끝]만큼 일정하게 바뀌어요. 중간 항들이 망원경처럼 상쇄되기 때문입니다.

망원경 상쇄 (telescoping)
// 경로 a -> b -> c 의 재가중 길이
w'(a,b) + w'(b,c)
= (w(a,b)+h[a]-h[b])
+ (w(b,c)+h[b]-h[c])
= w(a,b)+w(b,c) + h[a] - h[c]
             ↑ h[b] 상쇄됨!

// 결국 원래길이 + h[시작] - h[끝]
// 시작·끝이 같은 모든 경로가
// 똑같이 이동 → 최단 순위 불변!
왜 음수가 사라지나
// h[v]를 "가상정점->v 최단거리"로
// 잡으면, 삼각부등식에 의해

//   h[v] <= h[u] + w(u,v)
// 항상 성립 (최단거리의 성질)

// 이를 정리하면
//   w(u,v) + h[u] - h[v] >= 0
//   =  w'(u,v)  >= 0  ✓

// 모든 재가중 간선이 음수가 아님!
💡 두 마리 토끼 — 재가중은 ① 모든 간선을 0 이상으로 만들면서(다익스트라 사용 가능), ② 최단 경로의 순위는 그대로 유지합니다(정답 보존). 이 두 성질이 존슨을 가능하게 합니다.

포텐셜 h[]는 어떻게 구하나 — 벨만–포드

포텐셜 h[v]는 "가상 정점에서 v까지의 최단 거리"입니다. 가상 정점 0을 만들어 모든 정점으로 가중치 0 간선을 긋고, 거기서 벨만–포드를 돌립니다.

C — 포텐셜 계산
/* 가상정점 0 -> 모든 v 가중치 0 */
/* h[v] = 가상정점에서 v까지 최단 */
for (int i = 1; i <= n; i++)
    h[i] = 0;    /* 0-간선이라 초기 0 */

/* 벨만-포드: 정점 n+1개니 n번 반복 */
for (int r = 1; r <= n; r++)
    for (int e = 0; e < m; e++)
        if (h[eu[e]] + ew[e] < h[ev[e]])
            h[ev[e]] = h[eu[e]] + ew[e];
왜 가상 정점인가
// 어느 한 정점에서 시작하면
// 도달 못 하는 정점이 있을 수 있음

// 가상정점을 모든 곳에 0-간선으로
// 연결하면, 모든 정점이 도달 가능

// h[v]가 모두 유한한 값을 가짐
// → 재가중이 온전히 작동

// 가상정점은 나가는 간선만 있어
// 원래 최단경로엔 영향 없음
⚠️ 이 벨만–포드가 음수 사이클을 탐지하는 역할도 합니다. 만약 음수 사이클이 있으면 포텐셜이 수렴하지 않으므로, 존슨은 여기서 "음수 사이클 존재"를 보고하고 멈춥니다.

재가중 그래프에서 다익스트라 V번

이제 모든 간선이 0 이상이니 다익스트라를 쓸 수 있습니다. 각 정점을 출발점으로 V번 돌려, 재가중 그래프상의 거리 d'를 모두 구합니다.

출발점 s = –
재가중 그래프 거리 d′ (다익스트라 결과)
복원된 원래 거리 d = d′ − h[i] + h[j]
[다음 출발점 ▶]으로 각 정점에서 다익스트라를 돌려보세요.
💡 왼쪽(재가중 거리)은 모두 0 이상입니다(다익스트라가 만든 값). 오른쪽은 복원 공식을 적용한 진짜 최단 거리로, 음수도 나타납니다. 한 행씩 채워지는 걸 보세요.

거리 복원 공식

재가중 그래프에서 구한 거리 d'(u,v)를 원래 거리로 되돌립니다.

d(u, v) = d′(u, v) − h[u] + h[v]

재가중 때 경로 길이가 h[u] − h[v]만큼 더해졌으니, 그만큼 빼주면 원래 거리입니다. 망원경 상쇄 덕분에 어느 경로였든 이 공식 하나로 정확히 복원됩니다.

예: d(1,5) 복원
// 재가중 그래프에서 d'(1,5) = 0
// h[1] = 0, h[5] = -4

d(1,5) = d'(1,5) - h[1] + h[5]
       = 0 - 0 + (-4)
       = -4   // 원래 최단거리! ✓
C — 복원
for (int v = 1; v <= n; v++) {
    if (dp[v] >= INF)
        dist[s][v] = INF;
    else
        dist[s][v] = dp[v] - h[s] + h[v];
}
왜 정확한가 — 재가중은 시작·끝이 같은 모든 경로를 똑같이 h[u]−h[v]만큼 이동시킵니다. 그래서 재가중 그래프의 최단 경로 = 원래 그래프의 최단 경로이고, 거리만 공식으로 되돌리면 완벽히 일치합니다.

존슨 vs 플로이드–워셜

둘 다 음수 간선이 있는 그래프에서 모든 쌍 최단 거리를 구합니다. 선택은 그래프의 밀도에 달렸어요.

플로이드–워셜존슨
복잡도O(V³)O(V²logV + VE)
구성3중 for문벨만–포드 1 + 다익스트라 V
희소(E≈V)V³≈ V²logV (빠름)
조밀(E≈V²)V³ (간단)V³logV (느림)
구현 난이도매우 쉬움부품 3개 조립
💡 경계선 — 간선이 정점 제곱에 가까운 조밀한 그래프면 플로이드–워셜이 간단하고 충분합니다. 간선이 정점 수에 가까운 희소한 그래프(도로망, SNS 등 현실 그래프 대부분)면 존슨이 훨씬 빠릅니다.
