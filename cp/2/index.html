시작하기 — 참과 거짓

조건문은 "참이면 실행, 거짓이면 건너뛰기"입니다

1조건문이란?참/거짓 판단

조건문(conditional)은 어떤 조건이 참(True)인지 거짓(False)인지 판단해서
실행할 코드를 고르는 문장이에요. 프로그램이 "생각하고 선택하게" 만드는 핵심 도구입니다!

예: "점수가 60점 이상이면 합격, 아니면 불합격"

점수 >= 60 ?
참 (True) ↓
"합격" 출력
거짓 (False) ↓
"불합격" 출력

if 문 (기본)

조건이 참일 때만 실행 · 거짓이면 그냥 넘어감

1가장 단순한 if참이면 실행

if 문은 조건이 참일 때만 안쪽 코드를 실행해요. 거짓이면 아무 일도 안 하고 넘어가요.

C 언어파이썬
if (n > 0) {
    printf("양수");
}		 if n > 0:
    print("양수")
🖥 양수인지 확인하기
#include <stdio.h>

int main() {
    int n;
    scanf("%d", &n);
    if (n > 0) {           // 조건: n이 0보다 큰가?
        printf("%d은(는) 양수입니다\n", n);
    }
    printf("검사 끝\n");      // 조건과 무관하게 항상 실행
    return 0;
}
# 양수인지 확인
n = int(input())
if n > 0:                 # 조건: n이 0보다 큰가?
    print(f"{n}은(는) 양수입니다")
print("검사 끝")          # 조건과 무관하게 항상 실행
if (n > 0) {n이 0보다 크면 중괄호 안 실행, 아니면 건너뜀
"검사 끝"if 밖의 문장 — 조건과 상관없이 항상 실행됨
양수(7), 음수(-3), 0을 각각 넣어보며 차이를 관찰!

else는 "그렇지 않으면"이라는 뜻이에요. if 조건이 거짓일 때 else 블록이 실행돼요.

거짓 → else 블록
"불합격" 출력

if (score >= 60) {

else:
    print("불합격")

여러 조건을 위에서 아래로 차례로 검사

1여러 갈래로 나누기계단식 검사

조건이 3개 이상일 때는 else if(C) / elif(파이썬)를 써요.
위에서 아래로 차례차례 검사하다가 참인 조건을 만나면 거기만 실행하고 끝나요!

C — else if파이썬 — elif
if (s >= 90) g = 'A';
else if (s >= 80) g = 'B';
else if (s >= 70) g = 'C';
else g = 'F';		 if s >= 90: g = "A"
elif s >= 80: g = "B"
elif s >= 70: g = "C"
else: g = "F"
💡 순서가 중요해요! 위에서부터 검사하므로 큰 조건부터 써야 해요.
만약 s >= 70을 맨 위에 두면, 95점도 70 이상이라 C등급이 되어버려요! (위에서 먼저 걸림)
🖥 점수 → 등급 (A/B/C/D/F)
#include <stdio.h>

int main() {
    int s;
    scanf("%d", &s);
    char grade;
    if (s >= 90)      grade = 'A';
    else if (s >= 80) grade = 'B';
    else if (s >= 70) grade = 'C';
    else if (s >= 60) grade = 'D';
    else              grade = 'F';
    printf("%d점 -> %c등급\n", s, grade);
    return 0;
}
# 점수 → 등급
s = int(input())
if s >= 90:
    grade = "A"
elif s >= 80:
    grade = "B"
elif s >= 70:
    grade = "C"
elif s >= 60:
    grade = "D"
else:
    grade = "F"
print(f"{s}점 -> {grade}등급")

논리 연산자

여러 조건을 동시에 · 그리고(AND) · 또는(OR) · 부정(NOT)

1AND · OR · NOT조건 결합

조건을 여러 개 묶을 때 논리 연산자를 써요.

의미C 언어파이썬설명
그리고 (AND)&&and둘 다 참이어야 참
또는 (OR)||or하나라도 참이면 참
부정 (NOT)!not참↔거짓 뒤집기
💡 예시: "13살 이상 그리고 19살 미만이면 청소년"
• C: if (age >= 13 && age < 19)
• 파이썬: if age >= 13 and age < 19: (또는 13 <= age < 19도 가능!)
🐍 파이썬만의 편리함: 13 <= age < 19처럼 연속 비교가 돼요! C에서는 불가능하고 반드시 &&로 묶어야 해요.
🖥 나이로 연령대 판별 (AND 사용)
#include <stdio.h>

int main() {
    int age;
    scanf("%d", &age);
    if (age < 13) {
        printf("어린이\n");
    } else if (age >= 13 && age < 19) {
        printf("청소년\n");   // 13 이상 AND 19 미만
    } else {
        printf("성인\n");
    }
    return 0;
}
# 나이로 연령대 판별
age = int(input())
if age < 13:
    print("어린이")
elif 13 <= age < 19:   # 연속 비교! (파이썬 전용)
    print("청소년")
else:
    print("성인")
age >= 13 && age < 19C: 두 조건을 &&로 묶어 "동시에 만족"
13 <= age < 19파이썬: 연속 비교로 간결하게!
8, 15, 25 등 여러 나이로 연령대를 확인!

🖥 윤년 판별 (AND · OR 복합)
#include <stdio.h>

int main() {
    int y;
    scanf("%d", &y);
    // 4의 배수 AND (100의 배수 아님 OR 400의 배수)
    if ((y % 4 == 0 && y % 100 != 0) || y % 400 == 0) {
        printf("%d년은 윤년\n", y);
    } else {
        printf("%d년은 평년\n", y);
    }
    return 0;
}
# 윤년 판별
y = int(input())
# 4의 배수 and (100의 배수 아님 or 400의 배수)
if (y % 4 == 0 and y % 100 != 0) or y % 400 == 0:
    print(f"{y}년은 윤년")
else:
    print(f"{y}년은 평년")
2024(윤년), 2023(평년), 1900(평년!), 2000(윤년!)

그래서 1900년은 평년(100의 배수), 2000년은 윤년(400의 배수)! AND와 OR을 잘 조합한 대표 예제예요.
