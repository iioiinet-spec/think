1배열 vs 리스트, 핵심 차이고정 vs 가변

둘 다 여러 값을 순서대로 담는 그릇이에요. 하지만 결정적 차이가 있어요.

비교C 배열 (array)파이썬 리스트 (list)
크기선언 시 고정 (int a[10])자유롭게 늘고 줄음
자료형한 종류만 (모두 int)섞어도 됨 (int, str 혼합)
추가/삭제❌ 직접 구현해야 함✅ append/pop 내장

C는 내부에서 무슨 일이 일어나는지 직접 코드로 구현해봐요. 그러면 리스트를 더 깊이 이해하게 돼요!
2배열은 어떻게 생겼나?칸과 인덱스

배열/리스트는 번호(인덱스)가 붙은 칸들이 나란히 있는 구조예요. 인덱스는 0부터 시작해요!

arr[2]는 인덱스 2번 칸, 즉 세 번째 값인 30이에요. C와 파이썬 모두 [ ]로 접근하고 0부터 세는 규칙이 같아요!

⚠️ 인덱스 범위 주의! 길이 5인 배열의 유효 인덱스는 0~4예요.
C에서 arr[5]에 접근하면 엉뚱한 메모리를 건드려 위험해요. 파이썬은 list[5]면 친절히 오류를 알려줘요.
3C에서 "리스트처럼" 쓰려면?배열 + size

C 배열은 크기가 고정이라, 리스트처럼 쓰려면 넉넉한 배열 + 현재 크기 변수를 함께 관리해요.

🔵 C — 배열 + size
int arr[100]; // 넉넉하게 int size = 0; // 현재 개수 // 값 추가는 arr[size]에 넣고 // size를 1 늘리는 방식!
🟢 Python — 그냥 리스트
arr = [] # 빈 리스트 # 크기 관리 필요 없음! # 파이썬이 알아서 처리
💡 size 변수가 바로 파이썬 len()의 역할이에요. C에서는 우리가 직접 이 개수를 세고 관리해야 해요. 이게 리스트 내부의 비밀입니다! 🔑

기본 — 선언·접근·순회

배열/리스트 만들고, 값 읽고, 전체 훑기

1선언과 접근만들고 읽기
동작C 배열파이썬 리스트
만들기int a[5] = {10,20,30,40,50};a = [10,20,30,40,50]
값 읽기a[2] // 30a[2] # 30
값 바꾸기a[0] = 99;a[0] = 99
개수int n = 5; // 직접 관리len(a) # 5
마지막 원소a[n-1]a[-1] # 음수 인덱스!
💡 파이썬만의 편리함: a[-1]은 맨 뒤, a[-2]는 뒤에서 두 번째! C에는 음수 인덱스가 없어서 a[n-1]로 써야 해요.
🖥 배열 만들고 전체 출력 + 합계
#include <stdio.h>

int main() {
    int n;
    scanf("%d", &n);        // 개수
    int a[100];
    for (int i = 0; i < n; i++) {
        scanf("%d", &a[i]); // 값 입력
    }
    // 전체 순회하며 출력 + 합
    int sum = 0;
    for (int i = 0; i < n; i++) {
        printf("%d ", a[i]);
        sum += a[i];
    }
    printf("\n합계: %d\n", sum);
    return 0;
}
# 리스트 만들고 출력 + 합계
n = int(input())
a = list(map(int, input().split()))

for x in a:           # 리스트 직접 순회!
    print(x, end=" ")
print()
print("합계:", sum(a))  # sum() 내장 함수
C: for i<n인덱스로 접근 (a[i]), 개수 n을 직접 알아야 함
Py: for x in a원소를 직접 순회! 인덱스 없이 값이 바로 나옴
Py: sum(a)합계도 내장 함수 한 방! C는 직접 더해야 함
첫 줄=개수, 둘째 줄=값들 (공백 구분)

append · pop (끝에 추가/제거)

파이썬 한 줄 vs C 직접 구현 — size 관리가 핵심

1append — 맨 뒤에 추가arr[size++]

파이썬 list.append(x)는 맨 뒤에 값을 붙여요. C에서는 size 위치에 넣고 size를 1 늘려요.

append(60) — size=5 위치에 넣고 size=6으로
010
120

🔵 C — 직접 구현
// arr[size]에 넣고 size 증가 arr[size] = 60; size++; // 개수 1 늘리기
🟢 Python — 내장
a.append(60) # 끝! 크기 관리 자동
2pop — 맨 뒤 제거size--

파이썬 list.pop()은 맨 뒤 값을 꺼내서 반환해요. C에서는 size를 1 줄이기만 하면 돼요! (값은 그냥 무시)

🔵 C — 직접 구현
// 마지막 값을 꺼내고 size 감소 int last = arr[size - 1]; size--; // 개수 1 줄이기 // last에 꺼낸 값이 있음
🟢 Python — 내장
last = a.pop() # 꺼낸 값이 last에 담김
💡 pop이 빠른 이유: 맨 뒤는 그냥 size만 줄이면 되니 아주 빨라요! (다른 원소를 건드릴 필요 없음)
하지만 맨 앞을 빼는 pop(0)은 뒤 원소를 모두 앞으로 당겨야 해서 느려요. (다음 탭에서!)
🖥 append로 짝수만 모으기
#include <stdio.h>

int main() {
    int n;
    scanf("%d", &n);
    int result[100];
    int size = 0;      // 결과 배열의 현재 개수
    for (int i = 0; i < n; i++) {
        int x;
        scanf("%d", &x);
        if (x % 2 == 0) {   // 짝수면
            result[size] = x; // append 직접 구현
            size++;
        }
    }
    for (int i = 0; i < size; i++)
        printf("%d ", result[i]);
    printf("\n짝수 %d개\n", size);
    return 0;
}
# append로 짝수만 모으기
n = int(input())
nums = list(map(int, input().split()))

result = []           # 빈 리스트
for x in nums:
    if x % 2 == 0:      # 짝수면
        result.append(x)  # 내장 append

print(*result)         # 리스트 펼쳐 출력
print(f"짝수 {len(result)}개")
C: result[size]=x; size++;append의 내부 동작 — 넣고 개수 늘리기
Py: result.append(x)위 두 줄을 한 번에! 크기는 자동 관리
Py: print(*result)* 로 리스트를 펼쳐서 공백 구분 출력
첫 줄=개수, 둘째 줄=값들 → 짝수만 추출

insert · remove (중간 삽입/삭제)

원소를 밀어야 하는 핵심 — C로 구현하면 원리가 보인다

1insert — 중간에 끼워넣기뒤로 밀기

파이썬 list.insert(i, x)는 i 위치에 값을 끼워넣어요.
끼워넣으려면 그 자리부터 뒤 원소들을 모두 한 칸씩 뒤로 밀어야 해요!

insert(2, 99) — 인덱스 2에 99 삽입 (30,40,50이 뒤로 밀림)
010
120
299
330
440
550
🔵 C — 직접 구현
// 뒤에서부터 한 칸씩 밀기 for (int j = size; j > pos; j--) { arr[j] = arr[j - 1]; } arr[pos] = 99; // 빈자리에 삽입 size++;
🟢 Python — 내장
a.insert(2, 99) # 밀기를 파이썬이 알아서!
⚠️ 밀기 방향이 중요! 뒤에서부터(j=size → pos) 밀어야 값이 안 덮여요.
앞에서부터 밀면 값이 연쇄적으로 복사되어 망가져요. 이게 C로 직접 짜봐야 아는 포인트!
2remove / del — 삭제하고 당기기앞으로 당기기

삭제는 반대예요. 지운 자리의 뒤 원소들을 앞으로 한 칸씩 당겨서 빈틈을 메꿔요.

🔵 C — 직접 구현
// pos 뒤 원소를 앞으로 당기기 for (int j = pos; j < size - 1; j++) { arr[j] = arr[j + 1]; } size--; // 개수 줄이기
🟢 Python — 내장
del a[pos] # 위치로 삭제 # 또는 a.remove(99) # 값으로 삭제
💡 del a[2]는 인덱스 2를 삭제, a.remove(99)는 값 99를 찾아 삭제(첫 번째 것만!).
C에서는 값으로 삭제하려면 먼저 그 값의 위치를 찾은 뒤 당기기를 해야 해요.
🖥 특정 위치에 값 삽입하기
#include <stdio.h>

int main() {
    int n;
    scanf("%d", &n);
    int a[100], size = n;
    for (int i = 0; i < n; i++) scanf("%d", &a[i]);
    int pos, val;
    scanf("%d %d", &pos, &val);

    // insert(pos, val) 직접 구현
    for (int j = size; j > pos; j--) {
        a[j] = a[j - 1];   // 뒤로 밀기
    }
    a[pos] = val;          // 빈자리에 삽입
    size++;

    for (int i = 0; i < size; i++) printf("%d ", a[i]);
    printf("\n");
    return 0;
}
# 특정 위치에 값 삽입
n = int(input())
a = list(map(int, input().split()))
pos, val = map(int, input().split())

a.insert(pos, val)   # 밀기 자동!

print(*a)
C: j=size; j>pos; j--뒤에서부터 당겨야 값이 안 덮임 (핵심!)
Py: a.insert(pos,val)위 5줄을 한 줄로! 내부에선 똑같이 밀기가 일어남
1줄=개수, 2줄=값들, 3줄=위치 값 (예: 2 99)

탐색 · 정렬

index · in · sort · reverse — C로 직접 구현

1메소드 한눈에 비교파이썬 vs C
하고 싶은 것파이썬 (내장)C (직접 구현)
값 있는지x in afor 순회하며 == 비교
위치 찾기a.index(x)for 순회, 같으면 i 반환
개수 세기a.count(x)for 순회, ==면 cnt++
뒤집기a.reverse()양끝 swap (투 포인터)
정렬a.sort()버블/선택 정렬 직접

💡 파이썬은 이 모든 걸 한 단어로 하지만, 내부에서는 결국 C가 하는 것과 똑같은 반복문이 돌아요. 직접 구현해보면 "마법"이 아니라 "반복문"이란 걸 알게 돼요!
🖥 배열 뒤집기 (reverse) — 투 포인터
#include <stdio.h>

int main() {
    int n;
    scanf("%d", &n);
    int a[100];
    for (int i = 0; i < n; i++) scanf("%d", &a[i]);

    // reverse() 직접 구현 — 양끝 맞바꾸기
    int left = 0, right = n - 1;
    while (left < right) {
        int tmp = a[left];   // swap
        a[left] = a[right];
        a[right] = tmp;
        left++; right--;
    }
    for (int i = 0; i < n; i++) printf("%d ", a[i]);
    printf("\n");
    return 0;
}
# reverse — 내장
n = int(input())
a = list(map(int, input().split()))

a.reverse()        # 제자리 뒤집기
print(*a)

# 참고: a[::-1] 로 뒤집힌 새 리스트도!
left=0, right=n-1양끝에서 시작하는 두 포인터
swap + left++ right--양끝을 바꾸고 가운데로 좁혀감
Py: a.reverse()내부적으로 똑같은 swap이 일어남!
1 2 3 4 5 → 5 4 3 2 1

🖥 정렬 (sort) — 선택 정렬 직접 구현
#include <stdio.h>

int main() {
    int n;
    scanf("%d", &n);
    int a[100];
    for (int i = 0; i < n; i++) scanf("%d", &a[i]);

    // sort() 직접 구현 — 선택 정렬
    for (int i = 0; i < n - 1; i++) {
        int min = i;
        for (int j = i + 1; j < n; j++) {
            if (a[j] < a[min]) min = j;
        }
        int t = a[i]; a[i] = a[min]; a[min] = t;
    }
    for (int i = 0; i < n; i++) printf("%d ", a[i]);
    printf("\n");
    return 0;
}
# sort — 내장 (매우 빠른 Timsort)
n = int(input())
a = list(map(int, input().split()))

a.sort()          # 오름차순 정렬
print(*a)

# 내림차순: a.sort(reverse=True)
선택 정렬남은 것 중 최솟값을 찾아 맨 앞과 교환 반복
Py: a.sort()파이썬은 Timsort라는 아주 빠른 정렬 내장!
5 2 8 1 9 3 → 1 2 3 5 8 9
