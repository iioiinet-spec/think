1표준 입력과 표준 출력stdin / stdout

모든 프로그램은 입력(Input)을 받아 처리(Process)하고 출력(Output)을 내보내요.
키보드로 넣는 입력을 표준 입력(stdin), 화면에 나오는 출력을 표준 출력(stdout)이라고 합니다.

STDIN

STDOUT
🖥️ 출력

입력 3 5 → 처리 3+5 → 출력 8

2C와 파이썬의 입출력 함수한눈에 비교

출력printf("%d", x);print(x)
입력scanf("%d", &x);x = int(input())
헤더/준비#include <stdio.h>필요 없음
💡 가장 큰 차이: C의 scanf는 변수 앞에 &(주소 기호)가 필요하고, 자료형별 서식(%d, %f)을 써야 해요.
파이썬은 input()이 항상 문자열로 받으므로, 숫자로 쓰려면 int()나 float()로 바꿔줘야 합니다.

출력 (Output)

printf 서식 지정자 · print 옵션 · 줄바꿈 제어

1C의 printf 서식 지정자%d %f %c %s

C의 printf는 값의 자료형에 맞는 서식 지정자가 필요해요.

%d
정수 (int)
printf("%d", 42)
%f
실수 (float/double)
printf("%f", 3.14)
%c
문자 하나
printf("%c", 'A')
%s
문자열
printf("%s", "안녕")
%lld
큰 정수 (long long)
printf("%lld", n)
%.2f
소수점 2자리
printf("%.2f", 3.14159)
💡 \n은 줄바꿈이에요. C의 printf는 \n을 직접 넣어야 줄이 바뀝니다.
파이썬 print()는 자동으로 줄을 바꾸고, 안 바꾸려면 end=""를 붙여요.
🖥 출력 실습 — 여러 서식으로 출력하기
#include <stdio.h>

int main() {
    printf("정수: %d\n", 42);
    printf("실수: %.2f\n", 3.14159);
    printf("문자: %c\n", 'A');
    printf("문자열: %s\n", "세영코딩");
    printf("합: %d + %d = %d\n", 3, 5, 3+5);
    return 0;
}
# 파이썬은 서식 지정자 없이 간단하게!
print("정수:", 42)
print("실수:", round(3.14159, 2))
print("문자:", "A")
print("문자열:", "세영코딩")
print(f"합: {3} + {5} = {3+5}")  # f-string
이 예제는 입력이 없어요. 바로 실행!

입력 (Input)

scanf로 값 받기 · input()으로 값 받기 · 형변환

1한 개의 정수 입력받기scanf / int(input())
단계C 언어파이썬
① 변수 준비int n;필요 없음
② 입력받기scanf("%d", &n);n = int(input())
③ 사용printf("%d", n*2);print(n * 2)
⚠️ C에서 &를 꼭 붙이세요! scanf("%d", &n)의 &n은 "n의 주소"를 뜻해요. 이걸 빠뜨리면 프로그램이 오작동합니다.
⚠️ 파이썬 input()은 문자열이에요. 숫자 계산하려면 반드시 int()로 감싸야 해요!
🖥 입력 실습 — 수를 입력받아 2배 출력
#include <stdio.h>

int main() {

}
# 정수 하나 입력받아 2배 출력
n = int(input())   # 문자열을 정수로 변환
print(n * 2)

여러 값 입출력

한 줄에 여러 수 · split() · 여러 줄 입력

1한 줄에 여러 수 입력받기공백으로 구분

3 5 처럼 한 줄에 공백으로 구분된 여러 수를 받는 방법이에요. 코딩 문제에서 정말 자주 나와요!

방식C 언어파이썬
두 수 받기scanf("%d %d", &a, &b);a, b = map(int, input().split())
💡 파이썬의 마법 한 줄: map(int, input().split())
• input().split() → 공백으로 잘라서 리스트로 (예: ["3","5"])
• map(int, ...) → 각 조각을 정수로 변환
• a, b = ... → 두 변수에 나눠 담기
🖥 두 수를 입력받아 합·차·곱 출력
#include <stdio.h>

int main() {
    int a, b;
    scanf("%d %d", &a, &b);  // 두 수 입력
    printf("합: %d\n", a + b);
    printf("차: %d\n", a - b);
    printf("곱: %d\n", a * b);
    return 0;
}
# 두 수 입력받아 합·차·곱
a, b = map(int, input().split())
print("합:", a + b)
print("차:", a - b)
print("곱:", a * b)

2여러 줄로 여러 수 입력받기반복 입력

첫 줄에 개수 N, 다음 줄부터 N개의 수가 주어질 때 — 반복문으로 받아요.

🖥 N개의 수를 입력받아 총합 출력
#include <stdio.h>

int main() {
    int n;
    scanf("%d", &n);      // 개수 N
    int sum = 0, x;
    for (int i = 0; i < n; i++) {
        scanf("%d", &x);  // 수 하나씩 입력
        sum += x;         // 누적
    }
    printf("총합: %d\n", sum);
    return 0;
}
# N개의 수 입력받아 총합
n = int(input())        # 개수
total = 0
for i in range(n):
    x = int(input())    # 수 하나씩
    total += x
print("총합:", total)

# [고급] 한 줄로: sum(int(input()) for _ in range(n))
첫 줄=개수, 그 아래 줄마다 수 하나씩 (줄바꿈으로 구분)

자료형별 입출력

정수 · 실수 · 문자열을 정확하게 다루기

1자료형별 입력 방법int / float / string
자료형C — scanf파이썬 — input
정수scanf("%d", &n);n = int(input())
실수scanf("%lf", &x);x = float(input())
문자열scanf("%s", str);s = input()
문자 하나scanf(" %c", &ch);ch = input()[0]
⚠️ C 실수 입력은 %lf(double)를 써요. 출력은 %f인데 입력은 %lf인 점 주의!
💡 파이썬 문자열은 input() 그대로가 문자열이에요. 변환이 필요 없어요.
🖥 이름과 나이, 키를 입력받아 소개 출력
#include <stdio.h>

int main() {
    char name[50];
    int age;
    double height;
    scanf("%s", name);       // 문자열 (배열명엔 & 불필요)
    scanf("%d", &age);       // 정수
    scanf("%lf", &height);   // 실수 (%lf!)
    printf("%s님은 %d살, 키 %.1fcm\n",
           name, age, height);
    return 0;
}
# 이름, 나이, 키 입력 (각 줄)
name = input()          # 문자열 그대로
age = int(input())     # 정수 변환
height = float(input()) # 실수 변환
print(f"{name}님은 {age}살, 키 {height:.1f}cm")
char name[50]C 문자열은 char 배열 — 크기를 미리 정해야 함
scanf("%s", name)배열 이름은 이미 주소라 & 안 붙임
{height:.1f}파이썬 f-string에서 소수점 1자리 서식
