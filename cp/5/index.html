시작하기 — 딕셔너리란?

번호가 아니라 "이름표(키)"로 값을 찾는 자료구조

1리스트와 무엇이 다를까?번호 vs 이름표

리스트는 번호(인덱스)로 값을 찾아요. 딕셔너리는 이름표(키)로 값을 찾아요!

비교리스트 (list)딕셔너리 (dict)
찾는 방법a[0], a[1] (번호)d["사과"] (이름)
저장 형태값만 순서대로키:값 짝으로
예시[90, 85, 70]{"국어":90, "수학":85}
쓰는 때순서가 중요할 때이름으로 빠르게 찾을 때
💡 딕셔너리 = 사전! 영어 사전에서 단어(키)로 뜻(값)을 찾듯이,
d["apple"]로 "사과"를 찾아요. 번호를 몰라도 이름만 알면 바로 찾죠!
2딕셔너리는 어떻게 생겼나?키 → 값

딕셔너리는 키(key)와 값(value)이 짝을 이뤄요. 키로 값을 바로 찾을 수 있어요!

scores = {"국어": 90, "수학": 85, "영어": 70}
"국어"90
"수학"85
"영어"70

scores["수학"]이라고 쓰면 "수학"이라는 키에 연결된 값 85가 바로 나와요!

⚠️ 키는 중복될 수 없어요! 같은 키에 값을 또 넣으면 덮어써져요.
d["국어"]=90 후 d["국어"]=100이면 90은 사라지고 100만 남아요.
3C에는 딕셔너리가 없다!배열로 흉내

C에는 딕셔너리가 기본으로 없어요. 그래서 키 배열 + 값 배열을 나란히 두고,
키를 찾을 땐 처음부터 하나씩 비교(선형 탐색)해요.

🔵 C — 키/값 병렬 배열
char keys[100][20]; // 키들 int vals[100]; // 값들 int size = 0; // 개수 // "수학" 찾기 = keys를 훑어 // 같은 키의 위치 i를 찾고 vals[i]
🟢 Python — 딕셔너리
d = {} # 빈 딕셔너리 d["수학"] = 85 # 키로 바로 접근! # 탐색을 파이썬이 알아서
💡 이 페이지의 핵심: 파이썬이 d["수학"] 한 번에 찾는 걸,
C는 배열을 처음부터 훑어 키를 비교해야 해요. 이 "선형 탐색"이 딕셔너리 내부의 비밀입니다! 🔑
(진짜 딕셔너리는 '해시'로 더 빠르지만, 원리 이해엔 선형 탐색이 딱이에요.)

기본 — 만들기·접근·수정

키로 값을 넣고, 읽고, 바꾸기

1만들기와 접근d[key]
동작C (키/값 배열)파이썬 딕셔너리
만들기char k[100][20]; int v[100];d = {}
값 넣기strcpy(k[0],"수학"); v[0]=85;d["수학"] = 85
값 읽기find("수학") → v[i]d["수학"] # 85
값 수정v[i] = 100;d["수학"] = 100
개수size 변수len(d)
⚠️ 없는 키 접근 주의! 파이썬에서 d["없는키"]는 KeyError 오류가 나요.
안전하게 찾으려면 d.get("키")를 쓰면 없을 때 None을 돌려줘요. (다음 탭에서!)
🖥 과목별 점수 딕셔너리 만들기
#include <stdio.h>
#include <string.h>

int main() {
    char keys[100][20];
    int vals[100], size = 0;
    int n;
    scanf("%d", &n);
    for (int i = 0; i < n; i++) {
        scanf("%s %d", keys[size], &vals[size]);
        size++;        // d[key]=val 직접 구현
    }
    // 전체 출력
    for (int i = 0; i < size; i++)
        printf("%s: %d\n", keys[i], vals[i]);
    printf("과목 수: %d\n", size);
    return 0;
}
# 과목별 점수 딕셔너리
n = int(input())
d = {}                # 빈 딕셔너리
for _ in range(n):
    key, val = input().split()
    d[key] = int(val)  # 키:값 저장

for key, val in d.items():
    print(f"{key}: {val}")
print(f"과목 수: {len(d)}")
C: keys[]/vals[]키와 값을 두 배열에 나란히 저장
Py: d[key]=int(val)키:값 짝을 한 줄로! size 관리 자동
Py: d.items()키와 값을 짝으로 순회 (C는 인덱스로)
1줄=개수, 이후 각 줄=키 값 (공백 구분)

조회 · 존재확인

get · in · keys · values — C는 선형 탐색으로

1키 찾기의 핵심 — findKey()선형 탐색

C 딕셔너리의 모든 동작은 "키가 몇 번째에 있나?"를 찾는 데서 시작해요. 이걸 함수로 만들어요!

🔵 C — findKey 직접 구현
// 키를 찾아 위치 반환 (없으면 -1) int findKey(char target[]) { for (int i = 0; i < size; i++) { if (strcmp(keys[i], target) == 0) return i; // 찾음! } return -1; // 없음 }
🟢 Python — 필요 없음
# 파이썬은 내부에서 자동! d["수학"] # 바로 접근 "수학" in d # 존재 확인 # 탐색을 직접 짤 필요 없음
💡 strcmp(a,b)==0은 두 문자열이 같다는 뜻이에요. C에서 문자열 비교는 ==가 아니라 strcmp를 써야 해요! 이 findKey가 있으면 get·in·수정·삭제를 모두 만들 수 있어요.
2get · in — 안전하게 조회없어도 OK
하고 싶은 것파이썬C (findKey 활용)
값 조회d["수학"]vals[findKey("수학")]
안전 조회d.get("수학", 0)i<0 ? 0 : vals[i]
존재 확인"수학" in dfindKey("수학") >= 0
키 목록d.keys()keys[0..size-1]
값 목록d.values()vals[0..size-1]
💡 d.get("키", 기본값)은 키가 없어도 오류 대신 기본값을 돌려줘요. 빈도수 세기 같은 곳에서 아주 유용해요! (④번 탭에서 활용)
🖥 이름으로 점수 조회 (있으면 점수, 없으면 안내)
#include <stdio.h>
#include <string.h>

char keys[100][20]; int vals[100], size = 0;

int findKey(char t[]) {
    for (int i = 0; i < size; i++)
        if (strcmp(keys[i], t) == 0) return i;
    return -1;   // 없음
}
int main() {
    int n; scanf("%d", &n);
    for (int i = 0; i < n; i++) {
        scanf("%s %d", keys[size], &vals[size]); size++;
    }
    char q[20]; scanf("%s", q);
    int idx = findKey(q);   // 선형 탐색
    if (idx >= 0) printf("%s: %d\n", q, vals[idx]);
    else printf("%s 없음\n", q);
    return 0;
}
# 이름으로 점수 조회
n = int(input())
d = {}
for _ in range(n):
    key, val = input().split()
    d[key] = int(val)

q = input()
if q in d:            # 존재 확인
    print(f"{q}: {d[q]}")
else:
    print(f"{q} 없음")

# 한 줄로: print(d.get(q, f"{q} 없음"))
C: findKey(q)배열을 훑어 키 위치를 찾음 (없으면 -1)
Py: q in d한 단어로 존재 확인! 내부는 똑같은 탐색
마지막 줄에 찾을 이름. '과학'처럼 없는 것도!

추가 · 삭제

d[k]=v · del · pop — C는 있는지 먼저 확인

1추가 vs 수정 — 같은 문법!d[k]=v

파이썬에서 d[key]=val은 키가 없으면 추가, 있으면 수정이에요. C에서는 이 둘을 직접 나눠야 해요.

🔵 C — 직접 구현
int i = findKey(key); if (i >= 0) { vals[i] = val; // 있으면 수정 } else { strcpy(keys[size], key); vals[size] = val; size++; // 없으면 추가 }
🟢 Python — 내장
d[key] = val # 있으면 수정, 없으면 추가 # 알아서 판단!
⚠️ 키 중복 = 덮어쓰기! d["국어"]=90 후 d["국어"]=100이면
새 항목이 생기는 게 아니라 90이 100으로 바뀌어요. 딕셔너리 키는 항상 유일해요.
2del · pop — 키로 삭제찾고 당기기

삭제는 키를 찾은 뒤, 그 자리를 없애고 뒤 원소를 당겨요 (리스트 삭제와 같은 원리!).

🔵 C — 직접 구현
int i = findKey(key); if (i >= 0) { for (int j = i; j < size-1; j++) { strcpy(keys[j], keys[j+1]); vals[j] = vals[j+1]; // 당기기 } size--; }
🟢 Python — 내장
del d[key] # 키로 삭제 # 또는 v = d.pop(key) # 삭제+값 반환
💡 del d[k]는 그냥 삭제, d.pop(k)는 삭제하면서 그 값을 돌려줘요. 없는 키를 지우려 하면 둘 다 오류가 나니, d.pop(k, 기본값)으로 안전하게!
🖥 점수 추가/수정 후 특정 키 삭제
// findKey는 위와 동일하다고 가정
int main() {
    int n; scanf("%d", &n);
    for (int i = 0; i < n; i++) {
        char k[20]; int v;
        scanf("%s %d", k, &v);
        int idx = findKey(k);
        if (idx >= 0) vals[idx] = v;   // 수정
        else { strcpy(keys[size],k); vals[size]=v; size++; }
    }
    char del[20]; scanf("%s", del);
    int di = findKey(del);         // 삭제
    if (di >= 0) {
        for (int j = di; j < size-1; j++) {
            strcpy(keys[j],keys[j+1]); vals[j]=vals[j+1];
        }
        size--;
    }
    for (int i=0;i<size;i++) printf("%s:%d\n",keys[i],vals[i]);
    return 0;
}
# 추가/수정 후 삭제
n = int(input())
d = {}
for _ in range(n):
    k, v = input().split()
    d[k] = int(v)     # 추가 or 수정

target = input()
if target in d:
    del d[target]     # 삭제

for k, v in d.items():
    print(f"{k}:{v}")
C: findKey로 분기있으면 수정, 없으면 추가를 직접 나눔
Py: d[k]=v추가·수정을 자동 판단! 한 줄로 끝
C: 삭제 시 당기기뒤 원소를 앞으로 옮겨 빈틈 메움
국어가 90→100 수정됨에 주목! 마지막=삭제할 키

순회 · 활용

items 순회 · 빈도수 세기 — 딕셔너리의 대표 활용

1딕셔너리 순회키·값 짝으로
순회 방법파이썬C
키만for k in d:for i: keys[i]
값만for v in d.values():for i: vals[i]
키+값for k,v in d.items():for i: keys[i],vals[i]
💡 C에서는 결국 인덱스 i로 keys[i]와 vals[i]를 함께 보는 거예요. 파이썬 items()가 하는 일과 똑같아요!
🖥 글자 빈도수 세기 (딕셔너리의 대표 활용!)
#include <stdio.h>
#include <string.h>

int main() {
    char s[200]; scanf("%s", s);
    char keys[100]; int cnt[100], size = 0;
    for (int i = 0; s[i]; i++) {
        int idx = -1;
        for (int j = 0; j < size; j++)  // findKey
            if (keys[j] == s[i]) { idx = j; break; }
        if (idx >= 0) cnt[idx]++;       // 있으면 +1
        else { keys[size]=s[i]; cnt[size]=1; size++; }
    }
    for (int i=0;i<size;i++)
        printf("%c: %d\n", keys[i], cnt[i]);
    return 0;
}
# 글자 빈도수 — 딕셔너리 대표 예제!
s = input()
count = {}
for ch in s:
    if ch in count:
        count[ch] += 1    # 있으면 +1
    else:
        count[ch] = 1     # 없으면 1로 시작

for ch, c in count.items():
    print(f"{ch}: {c}")

# 더 짧게: count[ch] = count.get(ch, 0) + 1
"있으면 +1, 없으면 1"빈도수 세기의 핵심 패턴
Py: count.get(ch,0)+1get으로 없으면 0에서 시작 — 한 줄로!
C: 선형 탐색으로 findKey글자가 이미 있는지 매번 훑어 확인
banana → b:1 a:3 n:2. 다른 단어도 넣어보세요!
