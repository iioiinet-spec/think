🧰 준비물: 식을 코드로 옮기기

기하 공식은 곱하기(*)·나누기(/)·괄호만 알면 돼요. 수학 식을 그대로 코드로 옮기면 끝!

📘 계산 기호
곱하기 *, 나누기 /, 괄호 ( ) 로 계산 순서를 정해요.
예: (가로+세로)×2 → (w + h) * 2
CC 언어
// 변수에 값 담고 계산
int w = 5, h = 3;
int area = w * h;
printf("넓이=%d\n", area);  // 15
PyPython
# 변수에 값 담고 계산
w, h = 5, 3
area = w * h
print("넓이=", area)   # 15

🔺 별표(*)로 도형 그리기

반복문으로 별표를 찍으면 사각형·삼각형이 그려져요! 줄(행)과 칸(열)을 반복하는 거예요.

📘 이중 반복문으로 그리기

📐 각의 합과 빈 각 구하기

삼각형 세 각의 합은 180°. 두 각을 알면 나머지는 빼면 돼요. 다각형 각의 합도 공식으로!

📘 각의 합 공식
삼각형 빈 각 = 180 − (두 각의 합)
n각형 모든 각의 합 = (n − 2) × 180 (삼각형으로 쪼갠 개수 × 180)
🔬 코드 실험실 — 삼각형 빈 각 찾기
첫째 각 = 50°
둘째 각 = 70°

📘 Python의 별표 마법
Python은 "*" * 5 라고 하면 ***** 가 한 번에 나와요! 그래서 삼각형도 for i in range(1,n+1): print("*"*i) 처럼 아주 짧게 그릴 수 있어요.
⚠️ C의 나누기 함정
C에서 7 / 2 는 3이 돼요(소수 버림)! 삼각형 넓이처럼 소수가 필요하면 7.0 / 2 또는 double을 꼭 써요.
