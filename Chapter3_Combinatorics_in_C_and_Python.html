손으로 세던 경우의 수·순열·조합을 C와 Python으로! 값을 바꾸면 코드와 결과가 바로 바뀌어요.

🧰 준비물: 곱하기를 반복하기

조합론 코드의 핵심은 곱셈을 반복하는 거예요. 변수 하나에 계속 곱해 쌓으면 돼요!

📘 누적 곱셈
result = result * i 를 반복하면, 여러 수를 차례로 곱한 값이 쌓여요. 경우의 수는 거의 다 이 방식!
CC 언어
// 2 × 3 × 4 를 곱해 쌓기
int result = 1;
for (int i = 2; i <= 4; i++) {
    result = result * i;
}
printf("%d\n", result);  // 24
PyPython
# 2 × 3 × 4 를 곱해 쌓기
result = 1
for i in range(2, 5):
    result = result * i
print(result)   # 24

❗ 팩토리얼 n! 구하기

n명을 한 줄로 세우는 방법의 수 = n! = n × (n-1) × … × 2 × 1. 줄 세우기의 기본!

🏃 순열 nPr 구하기 (순서 있음)

n명 중 r명을 순서대로 뽑기 = n × (n-1) × … (r번 곱하기). 반장·부반장 뽑기처럼요!

📘 순열 공식
nPr = n부터 시작해 1씩 줄이며 r번 곱해요.
예: 5P2 = 5 × 4 = 20 (반장 5명 중, 부반장 남은 4명 중).

🤝 조합 nCr 구하기 (순서 없음)

n명 중 r명을 순서 상관없이 뽑기. 순열을 구한 뒤 중복된 순서(r!)로 나눠주면 돼요!

📘 조합 공식
nCr = nPr ÷ r!   (순서를 따진 것을, 순서 가짓수로 나눠 중복 제거)
예: 5C2 = (5×4) ÷ (2×1) = 20 ÷ 2 = 10.
🔬 코드 실험실 — nCr
전체 n = 5
뽑을 r = 2

⚠️ 조심: 수가 폭발해요
팩토리얼·순열은 아주 빠르게 커져요. 10! = 360만, 20!은 어마어마해요! C에서는 int로 13!만 넘어도 넘쳐버려요. 큰 수는 Python이 편해요.
