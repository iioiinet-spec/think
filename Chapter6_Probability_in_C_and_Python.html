확률·통계를 코드로 구하기 💻

합계·평균, 확률, 그리고 동전·주사위 실험까지! 코드로 수천 번 실험해봐요.

🧰 준비물: 더하기를 쌓기

통계의 시작은 '모두 더하기(합계)'예요. 변수 하나에 계속 더해 쌓으면 돼요!

📘 누적 덧셈
sum = sum + x 를 반복하면 여러 수의 합이 쌓여요. 평균은 이 합을 개수로 나누면 끝!
CC 언어
// 점수 3개 더하기
int score[] = {6, 8, 10};
int sum = 0;
for (int i = 0; i < 3; i++)
    sum = sum + score[i];
printf("합=%d\n", sum);  // 24
PyPython
# 점수 3개 더하기
score = [6, 8, 10]
total = sum(score)
print("합=", total)   # 24

💡 Python의 선물
Python은 sum(리스트) 한 줄로 합을 구해줘요! C는 직접 반복문으로 더해야 해요.

⚖️ 평균 구하기

평균 = 합계 ÷ 개수. 합을 구한 뒤 나누면 끝! 슬라이더로 점수를 바꿔봐요.

💡 소수 주의 (C)
평균은 소수가 될 수 있어요! C에서는 (double)sum / n 으로 나눠야 정확해요. Python은 그냥 / 면 돼요.

🎯 확률 계산하기

확률 = 원하는 경우 ÷ 전체 경우. 코드로는 나누기 한 번! 분수와 퍼센트로 보여줘요.

🔑 핵심
확률을 퍼센트로 바꾸려면 × 100! 예: 0.5 → 50%. 코드에서 p * 100 하면 돼요.

🪙 동전 실험 — 난수로 시뮬레이션

컴퓨터의 '랜덤(난수)'으로 동전을 수천 번 던져봐요! 많이 던질수록 50%에 가까워져요.

📘 난수란?
컴퓨터가 만드는 '아무 수'예요. C는 rand(), Python은 random을 써요. 0 또는 1을 뽑아 앞·뒤를 정해요!

📊 자료 세기 — 막대그래프 그리기

각 항목이 몇 개인지 세고, 그 수만큼 별표(*)를 찍으면 글자 막대그래프가 돼요!

📘 난수 준비 (C)
C에서 난수를 쓰려면 맨 위에 #include <stdlib.h> 와 srand(time(0)) 가 필요해요. Python은 import random 한 줄이면 끝!
⚠️ 평균 나누기 함정
C에서 정수끼리 나누면 소수가 사라져요(7/2 = 3)! 평균은 꼭 double로 나눠요.
