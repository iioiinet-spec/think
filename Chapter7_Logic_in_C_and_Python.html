🧰 참·거짓을 코드로 담기

코드에서도 참과 거짓을 값으로 다뤄요! Python은 True/False, C는 1/0을 써요.

📘 불(bool) 값
참·거짓을 나타내는 값을 불(boolean)이라고 해요.
Python: True / False
C: 1(참) / 0(거짓)
CC 언어
// 참은 1, 거짓은 0
int isSunny = 1;   // 참
int isRainy = 0;   // 거짓
printf("%d\n", isSunny);  // 1
PyPython
# 참은 True, 거짓은 False
is_sunny = True
is_rainy = False
print(is_sunny)   # True

두 수를 비교하면 참 또는 거짓이 나와요. 이게 모든 논리의 출발점이에요!

📘 비교 기호
== 같다, != 다르다, > 크다, < 작다. 같다는 = 가 아니라 == 두 개! (=는 값을 담는 거예요)

➡️ if 조건문 — 만약 ~라면

7장의 "만약 ~라면"이 바로 코드의 if문이에요! 조건이 참일 때만 안쪽이 실행돼요.

📘 if ~ else
if(조건): 조건이 참이면 실행
else: 거짓이면 대신 실행. "만약 비가 오면 우산, 아니면 그냥"처럼요!
