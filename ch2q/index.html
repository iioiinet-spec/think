11 GCD·LCM 관계 ★★★★☆
⬜
세 자연수 A, B, C에 대해 다음이 성립합니다.
GCD(A, B) = 6
GCD(B, C) = 10
GCD(A, C) = 15
LCM(A, B, C) = 900
A + B + C 의 최솟값을 구하시오.
GCD(A,B)=6 → 2×3 | A,B
GCD(B,C)=10 → 2×5 | B,C
GCD(A,C)=15 → 3×5 | A,C
→ B는 2와 3과 5의 배수, 즉 30의 배수!
LCM=900=2²×3²×5²를 이용해 A,B,C를 결정하세요.
12 소수 & 약수 ★★★★☆
⬜
어떤 자연수 N의 약수의 합이 N+1과 같을 때, N을 완전수에 가까운 수라 하지 않고 소수라고 합니다.
사실 소수 p의 약수의 합은 1 + p입니다.
약수의 합이 100 이하인 소수를 모두 구하고, 그 개수를 쓰시오.
(약수의 합 = 1 + p ≤ 100 → p ≤ 99)
소수 p의 약수의 합 = 1 + p
1 + p ≤ 100 → p ≤ 99
즉 99 이하의 소수 개수를 구하면 됩니다.
에라토스테네스의 체로 99 이하의 소수를 세어보세요!
13 복합 사고력 ★★★★☆
⬜
두 자리 자연수 N에 대해 N의 각 자리 숫자의 합이 N의 최대공약수 조건을 만족합니다.
N = 10a + b (a,b는 각 자리 숫자, a≥1)일 때,
GCD(N, a+b) = a+b 를 만족하는 두 자리 수를 모두 구하고, 그 개수를 쓰시오.
GCD(N, a+b) = a+b 이면 (a+b) | N
즉 N이 (각 자리 합)의 배수!
예: N=12, a+b=3, 12÷3=4 → ✓
N=10a+b, S=a+b일 때 S | (10a+b) 확인
10a+b = 10a+b, S=a+b → 10a+b ≡ 0 (mod S)
14 GCD·LCM 관계 ★★★★★
⬜
두 자연수의 최대공약수와 최소공배수의 합이 280이고,
최소공배수는 최대공약수의 정확히 9배입니다.
가능한 두 자연수의 쌍 (A, B) with A ≤ B를 모두 구하고, 쌍의 개수를 쓰시오.
GCD = g, LCM = 9g 로 놓으면
g + 9g = 280 → 10g = 280 → g = 28
LCM = 9×28 = 252
A = 28a', B = 28b', a'×b'=9, GCD(a',b')=1
9 = 1×9 (GCD=1 ✓) 또는 3×3 (GCD=3 ✗)
