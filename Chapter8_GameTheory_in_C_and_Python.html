게임을 코드로 만들기 ♟️

컴퓨터 상대 만들기, 님 게임 규칙과 이기는 전략, 동전 퍼즐까지 코드로 구현해요!

🧰 컴퓨터 상대 만들기 (난수)

게임의 상대가 되려면 컴퓨터가 '선택'을 해야 해요. 난수(랜덤)로 아무 수나 고르게 해요!

📘 난수로 고르기
범위 안에서 아무 수나 뽑아요.
C: rand() % 3 + 1 (1~3)
Python: random.randint(1, 3)
CC 언어
// 1~3 중 컴퓨터가 고르기
#include <stdlib.h>
#include <time.h>
srand(time(0));
int pick = rand() % 3 + 1;
printf("컴퓨터: %d개\n", pick);
PyPython
# 1~3 중 컴퓨터가 고르기
import random
pick = random.randint(1, 3)
print("컴퓨터:", pick, "개")

♟️ 님 게임 규칙을 코드로

막대가 몇 개 남았는지 변수로 관리하고, 가져간 만큼 빼면 돼요. 게임의 뼈대를 만들어요!

📘 게임의 흐름
① 막대 개수를 변수에 담기 → ② 번갈아 1~3개 빼기 → ③ 0이 될 때까지 반복(while) → ④ 마지막에 가져간 사람이 짐
🔬 코드 실험실 — 막대 빼기 시뮬레이션

한 번에 가져갈 수 = 2

while (막대 > 0) 로 막대가 다 없어질 때까지 반복! 이게 게임의 심장이에요.

🏆 이기는 전략을 코드로

8장에서 찾은 비밀! "상대에게 4의 배수+1을 남기기". 이걸 코드 한 줄로 계산해요.

📘 승리 공식
마지막 막대를 가져가면 지는 게임에서, 내 차례에 막대가 n개면
가져갈 개수 = (n − 1) % 4 로 계산! (이게 0이면 질 수밖에 없는 자리예요)

💡 왜 4일까?
나와 상대가 가져간 수를 합쳐 항상 4가 되게 맞추면(상대 1→나 3, 상대 2→나 2…), 4씩 줄어들어 결국 상대에게 1개를 남길 수 있어요!

📘 게임 = 규칙 + 반복 + 판단
게임 코드는 ① 상태를 변수로 두고 ② while로 반복하며 ③ if로 승패를 판단해요. 여기에 난수(랜덤 상대)나 전략(똑똑한 상대)을 넣으면 완성!
⚠️ 난수 준비 잊지 마요
C는 srand(time(0))를 한 번 해줘야 매번 다른 수가 나와요. 안 하면 항상 똑같은 수만! Python은 import random만 하면 돼요.
