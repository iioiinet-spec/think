CONCEPT약수와 배수부터 차근차근!
최대공약수와 최소공배수를 배우기 전에, 약수와 배수를 먼저 확인해요.

01약수란? 배수란?

약수 : 어떤 수를 나누어떨어지게 하는 수. 예) 12의 약수 → 1, 2, 3, 4, 6, 12

배수 : 어떤 수를 1배, 2배, 3배… 한 수. 예) 4의 배수 → 4, 8, 12, 16, …

최대공약수(GCD) = 두 수의 공통 약수 중 가장 큰 수 최소공배수(LCM) = 두 수의 공통 배수 중 가장 작은 수

02약수 탐험기 🔍

수를 넣으면 약수를 모두 찾아줘요. 약수는 짝을 지어 나온답니다! (예: 12 = 1×12 = 2×6 = 3×4)

INTERACTIVE · 1부터 100까지의 수를 넣어 보세요

03배수 생성기 🚀

INTERACTIVE · 배수를 10개까지 만들어 보세요
SEE공약수·공배수를 눈으로!
두 수의 약수와 배수를 나란히 놓으면, 공통인 것(초록)과 답(★주황)이 보여요.

01공약수 비교기 — 최대공약수 찾기

INTERACTIVE · 두 수를 바꿔 보세요

02공배수 비교기 — 최소공배수 찾기

INTERACTIVE · 배수를 나란히 놓고 처음 만나는 곳을 찾아요
METHOD거꾸로 나눗셈으로 한 번에!
두 수를 공약수로 계속 나누면, 최대공약수와 최소공배수를 한 번에 구할 수 있어요.

01방법 정리

① 두 수를 나란히 쓰고, 둘 다 나누어지는 수(공약수)로 계속 나눈다 ② 더 이상 같이 나눌 수 없으면 멈춘다 ③ 최대공약수 = 왼쪽에 쓴 수들을 모두 곱한 것 ④ 최소공배수 = 왼쪽 수들 × 마지막 남은 두 수까지 모두 곱한 것

02거꾸로 나눗셈 시뮬레이터 🪜

INTERACTIVE · 두 수를 넣으면 사다리를 자동으로 그려줘요

03신기한 관계 — 곱하면 같아요!

최대공약수 × 최소공배수 = A × B
INTERACTIVE · 위에서 넣은 두 수로 확인해 보세요
「관계 확인하기」를 누르면 시뮬레이터의 A, B로 계산해 보여줘요.
