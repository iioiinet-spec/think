1. 왜 점화식을 행렬로 바꿀까?

점화식은 앞의 값을 이용해 다음 값을 만드는 규칙입니다. 이 규칙을 행렬 하나로 표현하면, 여러 번의 계산을 행렬 거듭제곱 한 번으로 압축할 수 있습니다.

일반 반복

1번째부터 n번째까지 차례대로 계산합니다.

시간 복잡도: O(n)

행렬 표현

한 단계 이동 규칙을 전이행렬로 만듭니다.

Sₙ = M · Sₙ₋₁

빠른 거듭제곱

M을 분할정복으로 거듭제곱합니다.

시간 복잡도: O(log n)
핵심 생각: “다음 항을 계산하는 데 필요한 값들을 한 묶음으로 저장하자.” 이 묶음이 상태 벡터입니다.

2. 점화식을 행렬로 바꾸는 4단계

1단계. 다음 항에 필요한 이전 항의 개수를 찾는다

예를 들어 aₙ = 2aₙ₋₁ + 3aₙ₋₂는 직전 두 항이 필요합니다. 따라서 상태 벡터의 크기는 2입니다.

2단계. 상태 벡터를 만든다

Sₙ = aₙaₙ₋₁

첫 칸에는 지금 구하려는 항, 다음 칸에는 한 단계 이전 항을 놓습니다.

3단계. 다음 상태를 이전 상태의 선형결합으로 표현한다

aₙaₙ₋₁ = 2310 aₙ₋₁aₙ₋₂

첫째 행 [2, 3]은 점화식의 계수입니다. 둘째 행 [1, 0]은 aₙ₋₁을 다음 상태의 둘째 칸으로 그대로 옮깁니다.

4단계. 반복 적용을 거듭제곱으로 바꾼다

Sₙ = Mⁿ⁻¹ · S₁

초기 상태를 S₁로 잡았기 때문에, S₁에서 Sₙ까지 총 n-1번 이동합니다.

주의: 지수는 초기 상태를 어디로 잡았는지에 따라 달라집니다. 공식부터 외우지 말고 “몇 번 이동하는가?”를 세는 것이 안전합니다.

4. 예제 2: 세 항을 사용하는 점화식

aₙ = 2aₙ₋₁ + aₙ₋₂ + 3aₙ₋₃

다음 항을 만들 때 세 항이 필요하므로 상태 벡터의 크기는 3입니다.

aₙaₙ₋₁aₙ₋₂ = 213 100 010 aₙ₋₁aₙ₋₂aₙ₋₃

첫째 행

[2, 1, 3]은 점화식 계수입니다.

둘째 행

[1, 0, 0]은 첫째 값을 한 칸 아래로 이동합니다.

셋째 행

[0, 1, 0]은 둘째 값을 한 칸 아래로 이동합니다.

일반 규칙: k항 선형 점화식의 전이행렬은 k×k입니다. 첫째 행에는 점화식 계수를 적고, 그 아래에는 왼쪽 아래 방향으로 1을 배치합니다.

6-1. C++ 행렬 빠른 거듭제곱

#include <iostream>
using namespace std;

using ll = long long;
const ll MOD = 1000000007LL;

struct Matrix {
    ll a[2][2];
};

// 2×2 행렬 곱셈
Matrix multiply(const Matrix& x, const Matrix& y) {
    Matrix result = {{{0, 0}, {0, 0}}};

    for (int i = 0; i < 2; i++) {
        for (int j = 0; j < 2; j++) {
            for (int k = 0; k < 2; k++) {
                result.a[i][j] =
                    (result.a[i][j] + x.a[i][k] * y.a[k][j]) % MOD;
            }
        }
    }
    return result;
}

// 분할정복으로 matrix^exponent 계산
Matrix power(Matrix matrix, long long exponent) {
    // 단위행렬: 행렬 곱셈에서 숫자 1과 같은 역할
    Matrix result = {{{1, 0}, {0, 1}}};

    while (exponent > 0) {
        // 지수의 현재 비트가 1이면 결과에 곱한다.
        if (exponent % 2 == 1) {
            result = multiply(result, matrix);
        }

        // 밑을 제곱하고 지수를 절반으로 줄인다.
        matrix = multiply(matrix, matrix);
        exponent /= 2;
    }
    return result;
}

int main() {
    long long n;
    cin >> n;

    if (n == 0) {
        cout << 0;
        return 0;
    }

    // 피보나치 전이행렬
    Matrix transition = {{{1, 1}, {1, 0}}};

    // [F_n, F_(n-1)]^T = transition^(n-1) × [F_1, F_0]^T
    Matrix powered = power(transition, n - 1);

    // F_1 = 1, F_0 = 0이므로 첫째 행과 초기 벡터의 곱
    long long fibonacciN =
        (powered.a[0][0] * 1 + powered.a[0][1] * 0) % MOD;

    cout << fibonacciN;
}

M^(n-1)과 초기 벡터 [a₁, a₀]ᵀ를 곱하면 aₙ을 얻을 수 있습니다.

6-3. 어떤 점화식은 바로 행렬화할 수 없는가?

이 강의의 기본 방법은 선형 점화식에 적용됩니다. 예를 들어 aₙ = aₙ₋₁² + aₙ₋₂처럼 제곱이 들어가거나, aₙ = max(aₙ₋₁, aₙ₋₂)처럼 최댓값 연산이 들어가면 같은 방식의 일반 행렬곱으로 바로 표현하기 어렵습니다.
